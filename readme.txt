Project Name: 'Slice The Pie' - UCD Financials Guessing Game

## Description:
A web application game which allows users to guess the revenue and expenditure percentages of various sectors of UC Davis funding

## Features:
- Interactive guessing game
- Created to help students visualize where school money is coming and going
- Allows users learn more about their school in a fun way
- Provides real revenue and expenditure data to compare to user guesses
- Provides relevant statistics on UC Davis funding (pre 2022)
- Responsive design for mobile and desktop

## Technologies used:
- HTML/CSS
- Javascript
- D3
- React
- Vite
